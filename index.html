FREE SOFT / SERVICE

使えば業務が捗る便利なフリーソフト / サービスサイト

WinSCP

見やすく分かりやすい多機能なFTPソフトです。
サーバー上で編集できるテキストエディタの機能が付いているのも特徴です。

https://ja.osdn.net/projects/winscp/

FFFTP

シンプルな定番FTPソフトです。

https://ja.osdn.net/projects/ffftp/

FlieZilla

フォルダがツリー状になって表示されるため自分がどの階層にいるのか目視で確認でき、迷いにくいです。
世界で利用数が1番多いFTPソフトと言われています。

https://ja.osdn.net/projects/filezilla/

Gyazo

ドラッグ＆ドロップで指定範囲をキャプチャ出来るソフトです。キャプチャ後即WEBにアップロードされます。URLを共有すれば誰でも見ることができる便利なソフトです。ソフトをダウンロードすると静止画版と動画（Gifアニメ）版の2種類が付いてきます。ユーザー登録をすると過去の画像も見ることが出来ます。

https://gyazo.com/ja

Codic

コーディング時にクラス名を何にするか迷う時ありませんか？
そんな時、このサービスを使えば最適なクラス名を提案してくれます。

https://codic.jp/

Placehold.jp

モックアップに最適なシンプルなダミー画像を自動生成してくれるサイトです。日本語と英語の2言語対応しています。

http://placehold.jp/

PLACEHOLDER.COM

海外のモックアップ作成サイトその1。URLの後ろに任意のサイズを入力すると、指定したサイズの画像が取得できます。

https://placeholder.com/

Dynamic Dummy Image Generator

海外のモックアップ作成サイトその2。

https://dummyimage.com/

COMMONFOLDA

共有フォルダについて

共有フォルダとは

社員に共有したいファイルは、社内のサーバーにUPして共有します。 「HIROKEN共有」フォルダに、各社員それぞれ個人用のフォルダが割り当てられていますので、そちらにファイルをUPして下さい。

\\hiroken-fserver\HIROKEN共有

hiroken server

hiroken serverに色々なフォルダが格納されています。 基本的に使うフォルダは「HIROKEN共有」になります。 その他は状況に応じて使い分けて下さい。「フリー共有」もクリエイティブセクターは素材などを格納しているので使うことが多いです。

\\hiroken-fserver

制作物の場所

制作したサイトのデータなどはこちらに格納して下さい。自分以外の他の人でも閲覧・編集が可能になる様、作成に使用したPSDも「matereal」フォルダに入れて下さい。

\\hiroken-fserver\フリー共有\素材\Webサイトデータ\material

TEMPLATE

テンプレートについて

LPやサイトのHTMLデザインテンプレート

LPやサイトの予め枠組みが組まれたHTMLテンプレートがありますので、１からベースで作成するよりこちらを活用して下さい。

\\Hiroken-fserver\hiroken共有\クリエイティブセクター\共通テンプレ

ヒアリングシート

社内外問わず、依頼を受ける時は基本的にはヒアリングシートを元にヒアリングしてください。テンプレートがありますので、こちらを活用し、必要に応じて項目を編集して下さい。

\\hiroken-fserver\HIROKEN共有\クリエイティブセクター
